# CAP-014 --- Decision Pack Generator

## Purpose

Generate a complete Decision Pack ready for GitHub and Notion promotion.

## Inputs

-   Approved decisions
-   Architecture changes
-   Workflow changes

## Outputs

-   Markdown folder structure
-   CHANGELOG
-   README
-   Release summary
-   Git commit message
-   Notion promotion checklist

## Definition of Done

-   All required artifacts generated
-   Folder validates against Decision Pack standard
-   Ready to commit
